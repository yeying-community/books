# Agent 工程实战手册

这是一份面向工程师、产品负责人和 Agent 研究者的中文书稿，目标不是复述某个热门项目的宣传材料，而是把 Agent 从“能演示”拆回“能长期运行”的工程系统。

本书当前聚焦三类对象：

- `Claude Code`：重点看本地代码代理的运行时设计、工具契约、权限边界、上下文装配和长期任务续航。
- `OpenClaw`：重点看开源数字员工框架如何组织入口、执行器、技能和多 Agent 运行时。
- `Harness`：重点看把模型能力稳定交付出来时，运行时层到底承担了什么职责。

写作原则：

- 官方文档、公开仓库、可核验代码优先。
- 社区文章只作为线索，不作为事实本身。
- 先讲主链路，再讲局部技巧，避免写成功能目录。
- 尽量落到具体工程决策，而不是停留在抽象口号。

全书结构分为三部分：

- 理论与方法：先建立理解 Agent 的基础框架。
- 项目实践与对照：再看 `Claude Code`、`OpenClaw`、`Harness` 如何把这些问题落到工程里。
- 最佳实践与方法沉淀：最后收束成可复用的设计原则。

当前阶段先建立目录骨架和基础章节，后续再继续向下细化到 Prompt 装配、工具路由、权限系统、压缩机制、记忆系统和插件生态。
