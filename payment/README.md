# Payment 系列文档沉淀（草案 v0.1）

这个仓库用于沉淀 `payment` 系列知识库，目标是把“零散经验”整理为“可复用的方法体系”。

## 快速开始

1. 先阅读 [`docs/acquiring/00-入门导航-v1.0.md`](docs/acquiring/00-入门导航-v1.0.md)，进入收单入门主线。
2. 再阅读 [`docs/acquiring/00-发布导航-v2.2.md`](docs/acquiring/00-发布导航-v2.2.md)，查看最新发布版导航。
3. 然后阅读 [`docs/acquiring/README.md`](docs/acquiring/README.md)，查看完整目录与模板库。
4. 如需继续写作，阅读 [`docs/book-format.md`](docs/book-format.md) 与 [`docs/style-guide.md`](docs/style-guide.md)。
5. 新增章节时从 [`docs/templates/chapter-template.md`](docs/templates/chapter-template.md) 复制。

- `docs/book-format.md`: 整本书的推荐结构（篇章设计、章节节奏、质量门槛）。
