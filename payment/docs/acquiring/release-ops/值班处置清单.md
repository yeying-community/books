# 值班处置清单

> 版本：v1.7  
> 更新时间：2026-04-20  
> 适用范围：发布流水线异常、门禁判定异常、回滚异常

## 1. 初始确认（5分钟内）

- [ ] 确认告警来源和触发时间。
- [ ] 确认影响范围（版本、国家、商户、通道）。
- [ ] 确认当前任务状态（运行中/失败/回滚中）。
- [ ] 确认值班 owner 与备份 owner。

## 2. 分级判定

| 判定项 | 是 | 否 |
|---|---|---|
| 是否影响核心发布链路可用性 | 进入 `P0/P1` | 继续判定 |
| 是否存在自动回滚失败 | 触发人工接管 | 继续自动处置 |
| 是否影响资金/账务安全 | 升级 `P0` | 保持当前级别 |

## 3. 排查步骤

1. 检查门禁脚本返回码与错误日志。
2. 检查规则版本和发布版本是否一致。
3. 检查试点结论（GO/HOLD/ROLLBACK）是否异常。
4. 检查通知通道是否可用。
5. 检查回滚脚本执行状态与恢复结果。

## 4. 升级与通知

- [ ] 已在值班群同步事件级别与影响范围。
- [ ] 已通知发布 owner 与技术负责人。
- [ ] `P0/P1` 已拉起事故桥并记录时间线。
- [ ] 人工接管触发后已通知业务窗口。

## 5. 关闭标准

- [ ] 发布队列恢复无积压。
- [ ] 核心指标回归阈值内。
- [ ] 回滚与补偿动作执行完毕。
- [ ] 事件记录与复盘任务已创建。
