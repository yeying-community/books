## 3. 关键检查点

- [ ] 截止日前 7 天完成数据完整性检查
- [ ] 截止日前 3 天完成口径一致性复核
- [ ] 截止日前 1 天完成最终审批与签署
- [ ] 报送后 1 天完成回执归档

## 4. 逾期升级机制

| 逾期时长 | 升级对象 | 处置动作 |
|---|---|---|
| `T+1` | 责任人直属主管 | 立即补报并提交说明 |
| `T+2` | 合规负责人 | 启动专项复盘 |
| `T+3` | 管理层 | 启动应急治理会议 |
