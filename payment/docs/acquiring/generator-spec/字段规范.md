# 字段规范

## 1. 必填字段

| 字段 | 类型 | 示例 | 说明 |
|---|---|---|---|
| `doc_id` | string | `21` | 文档编号 |
| `title` | string | `图谱标准化指南` | 文档标题 |
| `version` | string | `v1.2` | 文档版本 |
| `owner` | string | `payment-docs` | 负责人 |
| `template_type` | string | `guide` | 模板类型 |
| `sections` | array | `["问题","先修知识","总结"]` | 章节块 |

## 2. 可选字段

| 字段 | 类型 | 示例 | 说明 |
|---|---|---|---|
| `audience` | array | `["产品","技术"]` | 目标读者 |
| `related_docs` | array | `["03","09"]` | 关联章节 |
| `extra_assets` | array | `["templates/..."]` | 关联模板 |
| `mode` | string | `safe_append` | 覆盖策略 |
