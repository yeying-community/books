# 图谱校验报告模板

> 版本：v1.3  
> 更新时间：2026-04-20  
> 用途：用于 CI 或发布门禁输出标准化校验结果

## 1. 报告头信息

| 字段 | 示例 | 说明 |
|---|---|---|
| 报告ID | `DV-REPORT-20260420-001` | 报告唯一编号 |
| 规则版本 | `R1.0` | 本次执行规则版本 |
| 执行时间 | `2026-04-20 21:30:00 +08:00` | 校验时间 |
| 执行范围 | `docs/acquiring/*.md` | 校验文件范围 |
| 触发来源 | `CI Pull Request` | 手工 / CI / 发布门禁 |

## 2. 汇总结果

| 指标 | 数值 |
|---|---|
| 扫描文件数 | `{{files_total}}` |
| Mermaid 图总数 | `{{diagram_total}}` |
| ERROR 数 | `{{error_total}}` |
| WARN 数 | `{{warn_total}}` |
| INFO 数 | `{{info_total}}` |
| 是否阻断发布 | `{{yes_or_no}}` |

## 3. 问题明细

| 严重级别 | 规则ID | 文件 | 位置 | 问题描述 | 修复建议 | 责任人 | 状态 |
|---|---|---|---|---|---|---|---|
| ERROR | `DV-001` | `23-图谱自动校验规范.md` | `L55` | Mermaid 语法错误 | 修复语法并重跑校验 | `@owner` | 待修复 |

## 4. 修复与复验记录

1. 初次执行结果：`ERROR={{x}} / WARN={{y}} / INFO={{z}}`。
2. 修复动作摘要：`{{修复说明}}`。
3. 复验结果：`ERROR={{x2}} / WARN={{y2}} / INFO={{z2}}`。
4. 结论：`{{通过发布门禁 / 阻断发布}}`。

## 5. 附件清单

- 原始校验输出日志：`{{path/to/raw-log}}`
