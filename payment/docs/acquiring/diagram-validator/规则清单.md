# 图谱校验规则清单

> 版本：v1.3  
> 更新时间：2026-04-20  
> 适用范围：`docs/acquiring/**/*.md` 内 Mermaid 图谱

## 1. 规则分级定义

| 级别 | 含义 | 处理策略 |
|---|---|---|
| `ERROR` | 阻断发布问题 | 必须修复后才能发布 |
| `WARN` | 中风险问题 | 可带风险发布，但必须记录 |
| `INFO` | 优化建议 | 可暂不处理 |

## 2. 核心规则（v1）

| 规则ID | 级别 | 规则说明 | 检测方式 | 修复建议 |
|---|---|---|---|---|
| `DV-001` | ERROR | Mermaid 代码块必须可解析 | 语法解析器检查 | 修复语法并重跑校验 |
| `DV-002` | ERROR | 图后必须包含输入/处理/输出三行说明 | 文本模式匹配 | 补齐三行说明 |
| `DV-003` | ERROR | 状态图状态名必须使用 `UPPER_SNAKE_CASE` | 正则检查状态名 | 统一状态命名 |
| `DV-004` | WARN | 节点名称不得使用无语义命名（如节点A/B） | 关键字匹配 | 用业务语义替换 |
| `DV-005` | WARN | 同章同术语命名不得不一致 | 术语比对 | 统一节点命名 |
| `DV-006` | INFO | 同章图方向建议统一（`LR` 或 `TD`） | 图头参数比对 | 统一方向设置 |
| `DV-007` | WARN | 图编号必须连续且在正文有引用 | 编号序列检查 | 修复编号或补正文引用 |

## 3. 场景扩展规则（可选）

| 规则ID | 级别 | 适用场景 | 规则说明 |
|---|---|---|---|
| `DV-101` | WARN | 清结算流程图 | 必须出现资金流向节点 |
| `DV-102` | WARN | 风控决策图 | 必须标注决策条件 |
| `DV-103` | INFO | 运营编排图 | 建议标注告警级别与升级路径 |

## 4. 误报反馈机制

1. 提交误报记录（规则ID、文件、上下文、建议处理）。
2. 规则维护人 48 小时内给出判定（误报/有效告警）。
3. 若确认为误报，追加白名单或修正规则实现。
4. 每周汇总误报率并更新规则版本。

## 5. 版本管理约定

- 规则版本号格式：`R{主版本}.{次版本}`，如 `R1.0`。
- 新增 `ERROR` 规则必须走评审流程。
- 每次变更必须附带变更说明和影响范围。
