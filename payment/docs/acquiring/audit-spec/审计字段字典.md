# 审计字段字典

> 版本：v1.8  
> 更新时间：2026-04-21  
> 适用范围：发布、门禁、审批、回滚、报表全链路

## 1. 字段分层

| 分层 | 说明 | 示例字段 |
|---|---|---|
| `base` | 基础元信息 | `event_id`、`event_time`、`env` |
| `actor` | 人员与角色信息 | `actor_id`、`actor_role` |
| `release` | 发布信息 | `release_version`、`target_scope` |
| `gate` | 门禁判定信息 | `gate_decision`、`error_count` |
| `approval` | 审批轨迹信息 | `approver_id`、`approve_result` |
| `ops` | 运维与回滚信息 | `rollback_flag`、`recover_result` |
| `report` | 报表与监管信息 | `report_version`、`reg_scope` |

## 2. 核心字段定义

| 字段名 | 类型 | 必填 | 枚举/格式 | 说明 |
|---|---|---|---|---|
| `event_id` | string | 是 | UUID/唯一ID | 事件唯一标识 |
| `event_time` | string | 是 | ISO-8601 | 事件时间戳 |
| `release_version` | string | 是 | `vX.Y` | 发布版本 |
| `rule_version` | string | 是 | `RX.Y.Z` | 规则版本 |
| `gate_decision` | enum | 是 | `GO/HOLD/ROLLBACK` | 门禁判定 |
| `approve_result` | enum | 否 | `PASS/REJECT` | 审批结论 |
| `rollback_flag` | bool | 是 | `true/false` | 是否触发回滚 |
| `incident_id` | string | 否 | 工单或事故号 | 故障关联 |
| `report_version` | string | 否 | `REP-X.Y` | 报表口径版本 |

## 3. 字段治理规则

1. 新增必填字段必须经过跨团队评审。
2. 删除字段必须保留 1 个版本过渡窗口。
3. 字段枚举变更必须同步报表口径模板。
4. 字段含义调整必须记录变更日期和影响范围。

## 4. 数据质量校验

| 校验项 | 规则 | 处理策略 |
|---|---|---|
| 完整性 | 必填字段不得为空 | 阻断归档 |
| 一致性 | `release_version` 与导航版本一致 | 阻断发布 |
| 合法性 | 枚举值必须在定义集合内 | 标记异常 |
| 可关联性 | `event_id` 可关联审批与回滚记录 | 触发补采集 |
