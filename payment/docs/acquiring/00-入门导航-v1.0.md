# 收单入门导航（v1.0）

> 版本：v1.0  
> 更新时间：2026-04-21  
> 维护人：payment-docs

## 1. 这份导航给谁看

- 第一次系统学习收单的产品、运营、技术同学。
- 需要先建立全局框架，再进入模板和自动化专题的读者。
- 目标是先掌握 80% 常见问题的定位方法，而不是一次读完整本书。

## 3. 每章怎么读最快

- 先看 `3分钟速读`，只抓核心结论。
- 再看 `主流程拆解`，理解支付链路。
- 最后看 `新手最容易错的 3 件事` 与 `Checklist`，用于落地自检。

## 4. 入门完成标准

- 能解释授权、清算、结算三者边界。
- 能区分争议预警和正式拒付处理路径。
- 遇到“支付成功未到账”时，能按交易层、清算层、结算层定位。
- 能说清风控策略如何影响结算周期和资金风险。

## 5. 建议节奏（5 天）

| 天数 | 阅读范围 | 输出物 |
|---|---|---|
| Day 1 | 01 + 02 | 角色边界图 + 进件检查项 |
| Day 2 | 03 | 生命周期状态图（交易/账务/资金） |
| Day 3 | 04 + 05 | 清算与结算差异清单 |
| Day 4 | 06 + 07 | 风险与争议闭环图 |
| Day 5 | 11 | 排障SOP与升级规则 |
