# 模板生成器输入输出契约

> 版本：v1.3  
> 更新时间：2026-04-20

## 1. 输入契约

### 1.1 必填字段

| 字段 | 类型 | 示例 | 说明 |
|---|---|---|---|
| `doc_id` | string | `24` | 文档编号 |
| `title` | string | `模板生成器原型实现方案` | 文档标题 |
| `version` | string | `v1.3` | 版本号 |
| `owner` | string | `payment-docs` | 文档负责人 |
| `template_type` | string | `chapter` | 模板类型 |
| `sections` | array | `["问题","先修知识","总结"]` | 生成章节块 |

### 1.2 可选字段

| 字段 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `audience` | array | `[]` | 读者角色 |
| `reviewer` | string | `TBD` | 审核人 |
| `appendix` | array | `[]` | 附录列表 |
| `force_blocks` | array | `[]` | 强制覆盖的章节块 |

### 1.3 策略字段

| 字段 | 可选值 | 说明 |
|---|---|---|
| `merge_mode` | `safe_append` / `force_replace` | 内容合并策略 |
| `validate_level` | `basic` / `strict` | 校验强度 |
| `release_gate` | `on` / `off` | 是否接入发布门禁 |

## 2. 输出契约

| 输出项 | 路径模式 | 说明 |
|---|---|---|
| 文档文件 | `docs/acquiring/{doc_id}-{title}.md` | 生成后的章节文档 |
| 生成日志 | `docs/acquiring/.logs/{timestamp}.json` | 输入参数与执行结果 |
| 校验报告 | `docs/acquiring/.reports/{timestamp}.md` | 结构/链接/图谱校验结果 |

## 3. 错误码约定

| 错误码 | 含义 | 处理建议 |
|---|---|---|
| `GEN-001` | 必填字段缺失 | 补齐输入参数并重试 |
| `GEN-002` | 模板类型不存在 | 修正 `template_type` |
| `GEN-003` | 文件写入冲突 | 检查覆盖策略或重命名 |
| `GEN-004` | 校验未通过（ERROR） | 修复问题后重跑 |
| `GEN-005` | 回滚失败 | 启用手动回滚方案 |

## 4. 兼容性约定

1. 新增字段必须保证旧输入可运行。
2. 删除字段必须提前一个版本公告。
3. `safe_append` 为默认策略，不得在无审批下切换为 `force_replace`。
4. 每次契约变更必须同步更新示例配置。

## 5. 输入示例

```json
{
  "doc_id": "24",
  "title": "模板生成器原型实现方案",
  "version": "v1.3",
  "owner": "payment-docs",
  "template_type": "chapter",
  "sections": ["本章问题", "先修知识", "总结"],
  "merge_mode": "safe_append",
  "validate_level": "strict",
  "release_gate": "on"
}
```
