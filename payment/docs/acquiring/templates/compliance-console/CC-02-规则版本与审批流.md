- 影响范围（国家/商户/支付方式）：

## 2. 审批流模板

1. 提单（产品/风控）
2. 合规审查
3. 法务审查
4. 技术发布审查
5. 生产发布

## 3. 审批SLA

| 节点 | 目标SLA | 超时动作 |
|---|---|---|
| 合规审查 | 24小时 | 升级L2 |
| 法务审查 | 24小时 | 升级L3 |
| 发布审查 | 12小时 | 暂停发布 |

## 4. 必填留痕字段

- 审批单号
- 审批人
- 审批时间（UTC）
- 审批结论
- 备注
