# 跨区域合规控制台模板目录（Compliance Console Templates）

> 版本：v0.8  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [CC-00-控制台主卡片.md](CC-00-控制台主卡片.md) 对齐目标与责任。
2. 使用 [CC-01-国家规则矩阵.md](CC-01-国家规则矩阵.md) 建立规则基线。
3. 通过 [CC-02-规则版本与审批流.md](CC-02-规则版本与审批流.md) 固化变更流程。
4. 用 [CC-03-数据驻留与跨境传输.md](CC-03-数据驻留与跨境传输.md) 管理数据边界。
5. 上线后按 [CC-04-合规告警与升级.md](CC-04-合规告警与升级.md) 和 [CC-05-审计留痕与周度复盘.md](CC-05-审计留痕与周度复盘.md) 运行。
