# CA-03 提交流水线与SLA

## 1. 提交流程

1. 证据包通过质量阈值。
2. 自动生成答辩封面与摘要。
3. 自动匹配提交窗口与通道要求。
4. 自动提交并回写提交结果。

## 2. SLA模板

| 阶段 | 目标时长 | 超时动作 |
|---|---|---|
| 案件入库 | 2小时内 | 自动告警 |
| 证据聚合 | 24小时内 | 升级人工 |
| 法务复核 | 12小时内 | 优先队列 |
| 对外提交 | 截止前48小时 | 强制升级 |

## 3. 阻断条件

- 证据评分低于阈值。
- 关键字段缺失（payment_id/ARN/reason_code）。
- 提交窗口已不足安全阈值。

## 4. 回执处理

- 提交成功：记录提交号与时间。
- 提交失败：按错误码分类重试或人工介入。
- 重复提交：按幂等键拦截。
