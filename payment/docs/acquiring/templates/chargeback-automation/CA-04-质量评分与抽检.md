# CA-04 质量评分与抽检

## 1. 评分维度

| 维度 | 分值 | 说明 |
|---|---|---|
| 完整性 | 30 | 关键证据是否齐全 |
| 一致性 | 30 | 金额、时间、单号是否一致 |
| 可验证性 | 20 | 是否可追溯到原始系统 |
| 可读性 | 20 | 时间线与陈述是否清晰 |

## 2. 自动提交阈值

- 总分 >= 85：可自动提交。
- 总分 70-84：需人工快速复核。
- 总分 < 70：禁止提交，退回补证。

## 3. 抽检机制

- 日抽检：自动提交案件抽检 10%。
- 周抽检：高金额案件抽检 100%。
- 专项抽检：败诉异常类案件全量抽检。

## 4. 问题闭环

- 抽检发现问题 -> 生成整改项 -> 指定Owner -> 跟踪关闭。
