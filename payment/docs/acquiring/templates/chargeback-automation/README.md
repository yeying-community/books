# 拒付自动化编排模板目录（Chargeback Automation Templates）

> 版本：v0.7  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [CA-00-编排主卡片.md](CA-00-编排主卡片.md) 明确目标与边界。
2. 配置 [CA-01-预警分流规则.md](CA-01-预警分流规则.md)。
3. 落地 [CA-02-取证流水线.md](CA-02-取证流水线.md) 与 [CA-03-提交流水线与SLA.md](CA-03-提交流水线与SLA.md)。
4. 上线后执行 [CA-04-质量评分与抽检.md](CA-04-质量评分与抽检.md)。
5. 每周使用 [CA-05-周度复盘模板.md](CA-05-周度复盘模板.md)。
