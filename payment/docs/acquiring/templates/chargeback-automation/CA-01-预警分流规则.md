# CA-01 预警分流规则

## 1. 分流维度

- 拒付原因码
- 案件金额区间
- 国家/地区
- 商户风险等级
- 历史拒付率

## 2. 分流规则模板

| 条件 | 分流队列 | 自动化等级 | 处理SLA |
|---|---|---|---|
| 欺诈类 + 高金额 + 高风险商户 | 队列A（高优） | 低（人工优先） | 4小时 |
| 履约类 + 中金额 | 队列B（标准） | 中（自动取证+人工复核） | 12小时 |
| 账单识别类 + 低金额 | 队列C（快处） | 高（自动提交） | 8小时 |

## 3. 兜底规则

- 无法识别原因码 -> 默认进入人工队列。
- 分流失败超过阈值 -> 触发P1告警并切人工全处理。
