# 通道路由策略模板目录（Routing Templates）

> 版本：v0.4  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [RT-00-策略主卡片.md](RT-00-策略主卡片.md) 明确目标和责任。
2. 再填写 [RT-01-国家币种通道矩阵.md](RT-01-国家币种通道矩阵.md) 确定可用通道池。
3. 使用 [RT-02-路由打分卡.md](RT-02-路由打分卡.md) 计算主备顺序。
4. 发布前填写 [RT-03-实时降级与故障切换.md](RT-03-实时降级与故障切换.md) 和 [RT-04-灰度发布与回滚.md](RT-04-灰度发布与回滚.md)。
5. 发布后一周填写 [RT-05-周度复盘模板.md](RT-05-周度复盘模板.md)。
