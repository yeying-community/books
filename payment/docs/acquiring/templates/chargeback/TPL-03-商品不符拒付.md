# 模板 03：商品不符拒付（Not as Described）

## 1. 适用场景

- 持卡人主张商品与描述不一致。
- 持卡人主张服务内容与购买页面承诺不一致。

## 2. 关键抗辩思路

- 证明商品详情页描述清晰且可核验。
- 证明发货或交付内容与订单一致。
- 证明商户提供了合理售后与补救方案。

## 3. 必备证据

| 编号 | 证据项 | 必须/可选 | 说明 |
|---|---|---|---|
| E01 | 下单页商品描述快照 | 必须 | 规格、型号、条款 |
| E02 | 订单确认页 | 必须 | 实际购买项与数量 |
| E03 | 出库/交付清单 | 必须 | 发货内容明细 |
| E04 | 客户投诉与客服处理记录 | 必须 | 投诉点与回复 |
| E05 | 售后方案证据 | 建议 | 换货、补发、退款提议 |
| E06 | 客户确认记录 | 可选 | 客户签字或确认邮件 |

## 4. 事实陈述模板

```text
客户下单页面明确展示 {关键规格/服务边界}（见 E01）。
订单 {merchant_order_id} 确认购买内容为 {item}（见 E02）。
商户实际交付内容与订单一致（见 E03）。
收到投诉后，商户于 {time_utc} 提供了 {售后方案}（见 E04/E05）。
综上，商户已按约定交付并提供合理售后处理。
```

## 5. 反例风险提示

- 商品描述版本无法回溯。
- 售后记录缺失或响应超时。
- 交付清单与订单存在明显差异。
