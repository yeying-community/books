# 拒付证据模板目录（Chargeback Templates）

> 版本：v0.3  
> 更新时间：2026-04-20

## 使用方式

1. 先复制 [TPL-00-拒付答辩封面.md](TPL-00-拒付答辩封面.md) 作为案件主文档。
2. 再按拒付类型选择一个模板补充证据。
3. 最后按 [../../12-拒付证据模板库.md](../../12-拒付证据模板库.md) 的提交清单做复核。
