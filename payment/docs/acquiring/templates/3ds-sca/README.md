# 3DS/SCA 模板目录（3DS SCA Templates）

> 版本：v0.6  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [DS-00-策略主卡片.md](DS-00-策略主卡片.md) 统一目标与边界。
2. 用 [DS-01-适用范围矩阵.md](DS-01-适用范围矩阵.md) 确定监管适用范围。
3. 用 [DS-02-豁免策略模板.md](DS-02-豁免策略模板.md) 配置豁免与挑战规则。
4. 上线前完成 [DS-03-认证流程与失败处理.md](DS-03-认证流程与失败处理.md)。
5. 监控与复盘使用 [DS-04-指标与告警.md](DS-04-指标与告警.md) 和 [DS-05-周度复盘模板.md](DS-05-周度复盘模板.md)。
