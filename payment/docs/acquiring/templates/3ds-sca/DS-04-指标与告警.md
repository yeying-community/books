# DS-04 指标与告警

## 1. 核心指标

| 指标 | 定义 | 目标值 | 刷新频率 |
|---|---|---|---|
| 认证触发率 | 进入3DS交易 / 总交易 | 按国家设定 | 小时 |
| 挑战率 | challenge交易 / 认证交易 | 按风险分层 | 小时 |
| 认证成功率 | 认证成功 / 认证交易 | >= 阈值 | 小时 |
| 授权通过率（认证后） | 认证后授权通过 / 认证成功 | >= 阈值 | 小时 |
| 拒付率（认证后） | 认证后拒付 / 清算成功 | <= 阈值 | 周 |

## 2. 告警阈值（示例）

- 认证成功率下降 > 8 个百分点（15分钟窗口）。
- 挑战超时率 > 5%（5分钟窗口）。
- 认证后授权通过率下降 > 6 个百分点（30分钟窗口）。
