# DS-03 认证流程与失败处理

## 1. 认证流程步骤

1. 发起认证请求。
2. 接收认证响应（Frictionless/Challenge）。
3. 认证结果回写交易状态。
4. 进入授权或失败处理。

## 2. 失败场景与处置

| 场景 | 典型错误 | 自动动作 | 人工动作 |
|---|---|---|---|
| 挑战超时 | timeout | 重试1次/切备通道 | 客服提示重试 |
| 认证失败 | auth_failed | 终止并拒绝 | 记录风控标签 |
| ACS不可用 | acs_down | 降级路由或延迟重试 | 拉起通道告警 |
| 数据缺失 | invalid_data | 拒绝并打回前端 | 排查字段映射 |
