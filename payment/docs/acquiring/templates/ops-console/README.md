# 支付运营自动化控制台模板目录（Ops Console Templates）

> 版本：v0.9  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [OC-00-控制台主卡片.md](OC-00-控制台主卡片.md) 对齐目标与职责。
2. 用 [OC-01-监控指标矩阵.md](OC-01-监控指标矩阵.md) 配置监控基线。
3. 用 [OC-02-告警与升级编排.md](OC-02-告警与升级编排.md) 定义告警动作。
4. 用 [OC-03-工单自动分派与Runbook.md](OC-03-工单自动分派与Runbook.md) 固化执行路径。
5. 值班与交接使用 [OC-04-值班排班与交接.md](OC-04-值班排班与交接.md)。
6. 故障后统一使用 [OC-05-事故复盘模板.md](OC-05-事故复盘模板.md)。
