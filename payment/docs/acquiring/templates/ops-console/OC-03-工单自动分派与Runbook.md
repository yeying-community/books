# OC-03 工单自动分派与Runbook

## 1. 工单分派规则

| 条件 | 目标队列 | 优先级 | 处理SLA |
|---|---|---|---|
| 支付成功率异常 | 支付运营队列 | P1 | 30分钟响应 |
| 通道错误率异常 | 技术值班队列 | P0 | 15分钟响应 |
| 结算延迟 | 结算运营队列 | P1 | 30分钟响应 |
| 拒付率异常 | 风控队列 | P1 | 2小时响应 |

## 2. Runbook模板

1. 异常确认（指标/日志/样本校验）
2. 快速止损（降级、限流、切备）
3. 根因定位（系统/通道/规则/数据）
4. 恢复验证（指标回升与样本复核）
5. 结案记录（时间线、动作、结果）
