# SE-04 出款与重试流程

## 1. 出款通道配置

| 通道 | 币种 | 时效 | 单笔上限 | 备注 |
|---|---|---|---|---|
| Bank-ACH | USD | T+1 | 50,000 | 默认 |
| Bank-SWIFT | 多币种 | T+1~T+3 | 200,000 | 跨境 |

## 2. 出款状态定义

- `PAYOUT_PENDING`
- `PAYOUT_SENT`
- `PAYOUT_FAILED`
- `RETRY_PENDING`
- `BANK_POSTED`
- `MANUAL_HOLD`

## 3. 重试策略（示例）

| 失败类型 | 自动重试 | 重试间隔 | 最大次数 | 人工介入 |
|---|---|---|---|---|
| 超时 | 是 | 10分钟 | 3 | 否 |
| 临时通道错误 | 是 | 30分钟 | 2 | 否 |
| 账户无效 | 否 | - | 0 | 是 |
| 合规拦截 | 否 | - | 0 | 是 |

## 4. 幂等规则

- 幂等键：`payout_id + attempt_no`
- 相同 attempt 禁止重复提交银行。
- 重试必须新增 attempt_no。
