# 结算引擎模板目录（Settlement Engine Templates）

> 版本：v0.5  
> 更新时间：2026-04-20

## 使用方式

1. 先填写 [SE-00-架构蓝图.md](SE-00-架构蓝图.md) 明确边界和组件。
2. 再填写 [SE-01-结算模板配置.md](SE-01-结算模板配置.md) 与 [SE-02-费率与准备金规则.md](SE-02-费率与准备金规则.md)。
3. 用 [SE-03-清分映射与账务分录.md](SE-03-清分映射与账务分录.md) 固化账务口径。
4. 上线前必须完成 [SE-04-出款与重试流程.md](SE-04-出款与重试流程.md)。
5. 故障后统一使用 [SE-05-异常回补与复盘.md](SE-05-异常回补与复盘.md)。
