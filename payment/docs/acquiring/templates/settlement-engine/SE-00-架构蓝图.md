## 2. 核心组件

| 组件 | 职责 | Owner |
|---|---|---|
| Batch Planner | 生成结算批次 | 结算技术 |
| Rule Engine | 命中费率/准备金规则 | 结算技术 |
| Ledger Writer | 写入账务分录 | 财务技术 |
| Payout Orchestrator | 出款编排与重试 | 资金技术 |
| Reconciliation | 对账与差异检测 | 财务运营 |

## 3. 数据流摘要

1. 读取清算净额数据。
2. 按商户模板生成批次与明细。
3. 计算费率、准备金、净额并写账。
4. 发起出款并回写状态。
5. 生成结算报表与对账差异任务。
